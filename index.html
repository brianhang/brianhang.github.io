Projects

  • TritonScheduler

    TritonScheduler is a tool written in Python for creating optimal schedules for students. The tool takes in a list of desired classes, then finds all the sections for those classes by scraping the UCSD Schedule of Classes. A genetic algorithm is then used to find an optimal schedule using a fitness function that considers the number of classes per day, the length of gaps between classes, and more.

  • CHIP8-Lua

    This project is a CHIP-8 interpreter written in Lua. The interpreter is enclosed in an object for easy embedding in other Lua applications. The project comes with an example that uses the interpreter and Love2D to play games.

  • ImageFabric

    ImageFabric is a program created with SFML and C++ that takes in an image and creates a fabric using the image. The fabric is simulated using Verlet integration. Here is a video demonstration.

  • JustinTime

    JustinTime is a work-in-progress 2D game created with SFML and C++. The game features several minigames that emulates the life of a generic college student.
